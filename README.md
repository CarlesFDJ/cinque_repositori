# cinque_repositori
El titol del nostre joc es diu DTC
El nostre joc està inspirat en el frogger, un videojoc on una granota ha de creuar carrers sense que sigui atropellada fins arribar al final, només té 3 vides, quan atropellen a la granota (main character) perd una vida, quan l'atropellen 3 cops mor, pot recuperar un cor menjant una mosca que hi ha pel camí.
